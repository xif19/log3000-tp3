# CALCULATRICE WEB - **Équipe 32**

## Objectif
Ce projet vise à développer et maintenir une application web de calculatrice simple. L'objectif principal est de structurer une base de code existante, d'implémenter de bonnes pratiques de gestion de versions, de documenter l'architecture logicielle et de mettre en place un pipeline de tests et de corrections de bogues fiable.

L'application est construite avec un backend Python avec Flask et une interface frontend en HTML/CSS.

## Prérequis d’installation
Avant de commencer, il faut avoir:

• Un compte GitHub.

• Git installé localement.

• Python et pip installés.

## Instructions d’installation

### 1. Cloner le dépot
```bash
git clone https://github.com/xif19/log3000-tp3.git
```

### 2. Créer un environnement virtuel dans le terminal
Windows: 
```bash
python -m venv venv
venv\Scripts\activate
```

Linux/MacOS:
```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Installer les dépendances
```bash
pip install flask
```

### 4. Démarrer l'application
```bash
python app.py
```

L'application va être accessible à l'addresse : http://127.0.0.1:5000/
